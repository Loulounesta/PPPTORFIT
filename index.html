Actions rapides
📋

Commencez par saisir les informations de la copropriété.

ORFIT, c'est ton carnet technique numérique ! 📋

Saisis tout sur place — murs, fenêtres, chauffage, photos DTG — et rentre au bureau avec un dossier béton.

🍎 iPhone / iPad — Safari 🤖 Android — Chrome 🖥️ PC / Mac 📡 Hors-ligne
Installation — à faire une seule fois
1
Installer sur l'écran d'accueil
Choisissez votre appareil
  1. Ouvrez Safari et tapez l'URL de l'appli
  2. Appuyez sur l'icône Partager ↑ (bas de l'écran)
  3. Appuyez sur « Sur l'écran d'accueil »
  4. Appuyez sur « Ajouter » → l'icône ORFIT apparaît 🎉
  1. Ouvrez Chrome et tapez l'URL de l'appli
  2. Appuyez sur les 3 points ⋮ en haut à droite
  3. Appuyez sur « Ajouter à l'écran d'accueil »
  4. Confirmez → l'icône ORFIT apparaît 🎉
  1. Ouvrez la page dans n'importe quel navigateur
  2. Mettez-la en favori (Ctrl+D / Cmd+D)
  3. Les données sont sauvegardées dans le navigateur
💡 Une fois installée, l'appli fonctionne sans connexion internet.

Pendant la visite
3
🚪 Créer les appartements
Onglet Appartements
  1. Appuyez sur « + Nouveau »
  2. Choisissez le N° échantillon (E01, E02…)
  3. Remplissez étage, surface, HSP, occupant
  4. Appuyez sur Enregistrer
💡 Utilisez Dupliquer si plusieurs appartements sont identiques.
4
🧱 🪟 🪨 ☁️ 🔑 Saisir les éléments
Murs · Fenêtres · Planchers · Plafonds · Portes

Depuis la fiche d'un appartement déplié, utilisez les boutons + Mur, + Fenêtre… pour aller directement au bon onglet.

💡 Les boutons − et + ajustent les dimensions par pas de 5 cm.

N'oubliez pas également de saisir les systèmes :

  • 🔥 Chauffage individuel — énergie, émetteur, marque, année, puissance
  • 🚿 ECS individuelle — type, énergie, marque, année, volume, catégorie
💡 Pour le chauffage et l'ECS collectifs, saisissez-les une seule fois dans l'onglet 🏢 Copropriété et appuyez sur ✓ Appliquer à tous les appartements.
5
📷 Photos DTG
Onglet DTG Photos
  1. Choisissez la catégorie (façades, chauffage, escaliers…)
  2. Tapez un titre court (ex : "Fissure façade Nord")
  3. Cochez les tags si besoin (Défaut, Urgence, Bon état…)
  4. Appuyez sur la grande zone 📷

L'appareil photo s'ouvre directement → « ✓ Enregistrer »

L'appareil photo s'ouvre → « ✓ Enregistrer »

L'explorateur Windows s'ouvre → sélectionnez une photo → « ✓ Enregistrer »

📸 Les photos sont dans l'application — pas dans la pellicule. Incluses dans tous les exports.

Questions fréquentes
❓ J'ai fermé l'app — ai-je perdu mes données ?
Non. Les données sont sauvegardées automatiquement. Rouvrez l'icône ORFIT.
❓ L'installation ne fonctionne pas sur iPhone ?
Utilisez Safari (pas Chrome ni Firefox). Seul Safari permet d'installer une PWA sur iPhone.
❓ Le bouton photo ne fonctionne pas sur tablette Windows ?
Sur Windows, l'explorateur de fichiers s'ouvre. Sélectionnez vos photos normalement.
❓ Je veux continuer sur un autre appareil ?
💾 → envoyez le .html par mail ou AirDrop → ouvrez dans Safari (iOS) ou Chrome (Android) → tout se restaure.
❓ Ça marche sans internet ?
Oui. Une fois installée, l'appli fonctionne entièrement hors-ligne.
