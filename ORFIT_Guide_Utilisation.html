ORFIT PPPT v2.0
Guide d'utilisation — Relevé terrain copropriété

📱 Bienvenue dans ORFIT PPPT

ORFIT, c'est ton carnet technique numérique ! 📋

Fini les feuilles volantes, les relevés illisibles et les photos perdues dans la pellicule. Tu saisies tout sur place — murs, fenêtres, chauffage, photos DTG — et tu rentres au bureau avec un dossier béton.

Fonctionne sur iPhone, Android et PC. Aucune installation depuis un store.

🍎 iPhone / iPad — Safari
🤖 Android — Chrome
🖥️ PC / Mac — tout navigateur
📡 Hors-ligne — fonctionne sans internet
Étape 1 — Installer l'application
1
Ouvrir l'application
iPhone · Android · PC · Mac

Ouvrez le navigateur de votre appareil et tapez l'adresse :

https://loulounesta.github.io/PPPTORFIT/
⚠️

iPhone/iPad : utilisez Safari (boussole bleue), pas Chrome.
Android : utilisez Chrome (cercle coloré), pas Samsung Internet.

2
Installer sur l'écran d'accueil
À faire une seule fois — comme une vraie app

Choisissez votre appareil :

  1. Une fois la page ouverte dans Safari, appuyez sur l'icône Partager ↑ (carré avec une flèche en bas de l'écran)
  2. Faites défiler le menu vers le bas
  3. Appuyez sur « Sur l'écran d'accueil »
  4. Appuyez sur « Ajouter » en haut à droite
  5. L'icône ORFIT apparaît sur votre bureau 🎉
  1. Une fois la page ouverte dans Chrome, appuyez sur les 3 points ⋮ en haut à droite
  2. Appuyez sur « Ajouter à l'écran d'accueil »
  3. Confirmez en appuyant sur « Ajouter »
  4. L'icône ORFIT apparaît sur votre bureau 🎉
💡

Sur certains Android, Chrome affiche directement une bannière en bas de l'écran : « Ajouter ORFIT PPPT à l'écran d'accueil ». Appuyez dessus.

  1. Sur PC ou Mac, ouvrez la page dans n'importe quel navigateur
  2. Mettez-la en favori (Ctrl+D ou Cmd+D) pour la retrouver facilement
  3. Les données sont sauvegardées dans le navigateur — ne changez pas de navigateur entre les sessions
💡

Sur PC, vous pouvez aussi utiliser la tablette Windows avec la caméra intégrée — un sélecteur de fichiers s'ouvrira pour choisir vos photos.

💡

Une fois installée, l'app fonctionne sans connexion internet. Idéal en immeuble avec mauvais réseau.

Appuyez sur Enregistrer. Les données sont sauvegardées automatiquement.

4
Renseigner les équipements collectifs
VMC, chauffage et ECS collectifs

Toujours dans l'onglet Copropriété, descendez pour remplir :

  1. Ventilation collective — choisissez le type de VMC dans la liste
  2. Chauffage collectif — renseignez énergie, générateur, marque, année → appuyez sur « ✓ Appliquer à tous les appartements »
  3. ECS collective — même procédure → « ✓ Appliquer à tous les appartements »
💡

En appuyant sur Appliquer à tous, tous les appartements créés seront automatiquement renseignés avec ces équipements collectifs. Gain de temps considérable.

Étape 3 — Pendant la visite
5
Créer les appartements
Onglet Appartements

Pour chaque logement visité :

  1. Appuyez sur « + Nouveau » en haut à droite
  2. Choisissez le N° échantillon (E01, E02…) dans le menu déroulant
  3. Remplissez étage, position, surface hab., HSP, occupant
  4. Appuyez sur Enregistrer

L'appartement apparaît dans la liste. Tapez dessus pour le déplier et voir le détail.

💡

Utilisez le bouton Dupliquer si plusieurs appartements sont identiques — modifiez ensuite seulement ce qui diffère.

6
Saisir les éléments du logement
Murs · Fenêtres · Plancher · Plafond · Porte

Depuis la fiche d'un appartement déplié, utilisez les boutons rapides :

BoutonCe qu'il fait
+ MurAller saisir un mur (orientation, matière, dimensions)
+ FenêtreAller saisir une fenêtre (type, matière, vitrage)
+ PlancherAller saisir le plancher bas
+ PlafondAller saisir le plafond
+ PorteAller saisir la porte palière
💡

Les boutons − et + ajustent les dimensions par pas de 5 cm. Rapide et précis, même avec des gants.

7
Prendre les photos DTG
Onglet 📷 DTG

Pour photographier les désordres et équipements :

  1. Allez dans l'onglet 📷 DTG
  2. Choisissez la catégorie (façades, chauffage, escaliers…)
  3. Tapez un titre court (ex : "Fissure façade Nord")
  4. Cochez les tags si nécessaire : Défaut, Urgence, Vétusté, Bon état…
  5. Appuyez sur la grande zone 📷

L'appareil photo s'ouvre directement. Prenez la photo puis validez. Appuyez sur « ✓ Enregistrer ces photos ».

L'appareil photo s'ouvre directement (ou un choix entre caméra et galerie). Prenez la photo puis validez. Appuyez sur « ✓ Enregistrer ces photos ».

L'explorateur de fichiers Windows s'ouvre. Sélectionnez une photo existante. Appuyez sur « ✓ Enregistrer ces photos ».

📸

Les photos sont stockées dans l'application — elles n'apparaissent pas dans votre pellicule. Elles sont incluses dans tous les exports.

Étape 4 — Sauvegarder et exporter
8
Sauvegarder régulièrement
Bouton 💾 — visible sur tous les onglets

Le bouton 💾 en haut à droite est toujours visible. Il génère un fichier ORFIT_NomCopro_date.html contenant toutes les données et photos.

  1. Appuyez sur 💾
  2. Le fichier se télécharge dans vos Fichiers (iOS) ou Téléchargements (Android/PC)
  3. Envoyez-le par mail ou AirDrop pour le partager ou l'archiver
⚠️

Sauvegardez après chaque appartement. En cas de batterie morte ou redémarrage, vous retrouvez toutes vos données en rouvrant le fichier .html sauvegardé.

9
Reprendre une visite sur un autre appareil
Transfert iPhone → PC ou Android → iPhone
  1. Appuyez sur 💾 sur l'appareil d'origine
  2. Envoyez le fichier .html par mail, AirDrop ou WhatsApp au destinataire
  3. Sur le nouvel appareil : ouvrez le fichier dans Safari (iOS) ou Chrome (Android)
  4. Toutes les données et photos se restaurent automatiquement ✓

Questions fréquentes

❓ J'ai fermé l'app par erreur — ai-je perdu mes données ?
Non. Les données sont sauvegardées automatiquement à chaque saisie. Rouvrez l'icône ORFIT sur votre écran d'accueil et tout est là.

❓ Ça ne marche pas sur mon iPhone — l'icône ne s'installe pas ?
Vérifiez que vous utilisez bien Safari (pas Chrome, pas Firefox). Sur iPhone, seul Safari permet l'installation sur l'écran d'accueil.

❓ Le bouton photo ne lance pas l'appareil photo sur Android ?
Vérifiez que Chrome a l'autorisation d'accéder à la caméra : Réglages Android → Apps → Chrome → Autorisations → Appareil photo → Autoriser.

❓ L'application est lente avec beaucoup de photos ?
Les photos sont stockées dans la mémoire de l'appli (IndexedDB). Pour libérer de la place, exportez le ZIP puis supprimez les photos via le bouton ✕ dans la galerie DTG.

❓ Je veux transmettre le dossier à un collègue ?
Appuyez sur 💾 → envoyez le fichier .html par mail ou AirDrop → votre collègue l'ouvre dans Safari (iOS) ou Chrome (Android) → toutes les données et photos s'affichent immédiatement.

❓ Est-ce que ça marche sans connexion internet ?
Oui. Une fois installée sur l'écran d'accueil, l'application fonctionne entièrement hors-ligne. Aucune connexion réseau n'est nécessaire pendant la visite.
